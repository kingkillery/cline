- **Description**: Autonomous coding agent right in your IDE, capable of creating and editing files, running commands, using the browser, and more with user approval at each step.

- `npm run compile` - typecheck, lint, and bundle the extension

- `npm run lint` - run Biome lint and proto lint checks
- `npm run test` - run unit and integration tests
- `npm run e2e` - run Playwright end-to-end tests
- `npm run ralph:run` - run the repo Ralph fresh-context loop
